使用 Grok 为 ClawMate 提供图像生成服务

> 本文档介绍如何注册 Grok 账号、部署 grok2api 反向代理服务，并将其接入 ClawMate 作为图像生成后端。
> 
> 使用的两个开源项目：
> 
> https://github.com/BytePioneer-AI/clawmate
> 
> https://github.com/chenyme/grok2api
> 
> 如果这些项目对你有帮助，请给他们一个Star！⭐

## 前置要求

  * • 已安装 OpenClaw
  * • 已安装 Docker 和 Docker Compose（推荐）或 Python 环境（本地开发）
  * • 可访问 Grok 官网的网络环境

## 第一步：注册 Grok 账号

### 1.1 访问注册页面

访问 Grok 官网：https://grok.com/

点击页面右上角的注册按钮。

### 1.2 选择注册方式

**推荐方式** ：使用 Google 账号注册（最快捷）

**备选方式** ：点击 "Sign up with your email" 使用其他邮箱注册

## 第二步：获取认证 Token

注册并登录 Grok 后，需要提取浏览器 Cookie 中的认证信息。

### 2.1 打开开发者工具

  1. 1\. 在 Grok 网站页面按 `F12` 打开浏览器开发者工具
  2. 2\. 刷新网页
  3. 3\. 切换到 **Network** 标签页
  4. 4\. 在过滤器中选择 **Fetch/XHR**

###  2.2 提取 SSO Token

  1. 1\. 点击任意一个网络请求
  2. 2\. 在右侧面板找到 **Request Headers** 部分
  3. 3\. 找到 `Cookie` 字段
  4. 4\. 复制其中的 `sso` 字段值（完整的 token 字符串）

  
提取 Token

> **注意** ：请妥善保管此 Token，不要泄露给他人。

## 第三步：部署 grok2api 反向代理

grok2api 是一个开源项目，可以将 Grok 的 API 转换为 OpenAI 兼容格式。

项目地址：https://github.com/chenyme/grok2api

### 方式一：Docker Compose 部署（推荐）
    
    
    # 克隆项目  
    git clone https://github.com/chenyme/grok2api  
    cd grok2api  
      
    # 启动服务  
    docker compose up -d

服务将在 `http://localhost:8000` 启动。

### 方式二：本地开发部署
    
    
    # 克隆项目  
    git clone https://github.com/chenyme/grok2api  
    cd grok2api  
      
    # 安装依赖并启动  
    uv sync  
    uv run main.py

## 第四步：配置 grok2api

### 4.1 访问管理后台

打开浏览器访问：http://127.0.0.1:8000/admin

使用默认密码登录：`grok2api`

### 4.2 修改管理密码（推荐）

登录后，在 **配置管理** 中修改默认密码和 API 密钥，提高安全性。

### 4.3 添加 Grok Token

  1. 1\. 在管理后台找到 **Token 管理** 页面
  2. 2\. 点击 **添加 Token**
  3. 3\. 粘贴第二步获取的 `sso` 字段值
  4. 4\. 保存配置

  

> **提示** ：额度显示可能不准确，如果实际有额度但显示为 0，可以手动设置额度值。

## 第五步：测试 API 服务

使用 curl 测试 API 是否正常工作：
    
    
    # 如果设置了 API 密钥，需将 $GROK2API_API_KEY 替换为真实密钥  
    curl http://localhost:8000/v1/chat/completions \  
      -H "Content-Type: application/json" \  
      -H "Authorization: Bearer $GROK2API_API_KEY" \  
      -d '{  
        "model": "grok-4",  
        "messages": [{"role":"user","content":"你好"}]  
      }'

## 第六步：接入 ClawMate

### 6.1 关于 ClawMate

ClawMate 是一个为 OpenClaw 提供视觉伴侣功能的插件，可以根据上下文、情绪和时间状态生成个性化的角色自拍。

**核心功能** ：

  * • **时间感知** ：根据当前时间（早晨/工作/傍晚/夜晚）调整场景和服装
  * • **主动发图** ：可配置角色主动发送自拍
  * • **多角色支持** ：内置多个角色，支持自定义
  * • **情境适配** ：根据对话内容生成相应的场景和表情

**应用场景** ：个人伴侣、虚拟导师、智能客服、专业顾问

**项目地址：https://github.com/BytePioneer-AI/clawmate**

###  6.2 配置 grok2api 图片格式

在 grok2api 管理后台的 **配置管理** 中，将图片格式设置为 **Base64** 。

### 6.3 安装 ClawMate

确保已安装 OpenClaw，然后运行一键安装命令：
    
    
    npx github:BytePioneer-AI/clawmate

交互式安装向导会引导你完成以下配置：

  * • 角色选择
  * • 主动发图配置
  * • 图像生成服务配置

### 6.4 配置图像生成服务

在安装向导的 **"选择图像生成服务"** 步骤中，选择 **"OpenAI 兼容接口"** 。

按照提示输入以下配置：

配置项| 值| 说明  
---|---|---  
**API Key**|  任意字符串| 如果 grok2api 未设置 API 密钥验证，可随意输入  
**Base URL**| ` http://localhost:8000/v1`| grok2api 服务地址  
**模型名称**| ` grok-imagine-1.0-edit`| 用于图像编辑的模型  

### 6.5 启动 ClawMate

配置完成后，重启 OpenClaw 使插件生效：
    
    
    # 重启 OpenClaw  
    openclaw restart

###  常见问题

### Q1: grok2api 显示额度为 0 但实际有额度怎么办？

在 Token 管理页面手动设置额度值即可。

### Q2: 如何修改 grok2api 的端口？

编辑 `docker-compose.yml` 或启动脚本中的端口配置，然后重启服务。

### Q3: grok2api 支持哪些 Grok 模型？

grok2api 支持以下 Grok 模型：

模型名| 计次| 可用账号| 对话| 图像| 视频  
---|---|---|---|---|---  
`grok-4`| 1| Basic/Super| ✓| ✓| -  
`grok-4-mini`| 1| Basic/Super| ✓| ✓| -  
`grok-4-thinking`| 1| Basic/Super| ✓| ✓| -  
`grok-4-heavy`| 4| Super| ✓| ✓| -  
`grok-4.1-mini`| 1| Basic/Super| ✓| ✓| -  
`grok-4.1-fast`| 1| Basic/Super| ✓| ✓| -  
`grok-4.1-expert`| 4| Basic/Super| ✓| ✓| -  
`grok-4.1-thinking`| 4| Basic/Super| ✓| ✓| -  
`grok-4.20-beta`| 1| Basic/Super| ✓| ✓| -  
`grok-imagine-1.0`| -| Basic/Super| -| ✓| -  
`grok-imagine-1.0-edit`| -| Basic/Super| -| ✓| -  
`grok-imagine-1.0-video`| -| Basic/Super| -| -| ✓

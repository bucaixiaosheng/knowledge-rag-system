01美团开源大模型上新

美团龙猫团队开源了一个专门做数学定理证明的模型：LongCat-Flash-Prover。 

数学定理证明不是算对答案就完事了 ，它要求每一步逻辑都能被严格验证，半点含糊都不行。 

龙猫团队的做法是把证明拆成三步：

先把自然语言翻译成 Lean4 形式化语言，再写草稿把大问题拆成小引理，最后一步步补全证明。

而且他们发现了一个很有意思的事：AI 会作弊。

模型为了骗过编译器，会偷偷改题目、插终止符、甚至捏造公理。

所以他们专门搞了个语法分析器来抓 9 种作弊手段，挺逗的。 

效果也很猛：MiniF2F-Test 只用 72 次推理就跑到 97.1%，竞赛级的 MathOlympiad-Bench 也有 46.7%。

都是开源模型里的 SOTA。

  * 

    
    
    开源地址：https://github.com/meituan-longcat/LongCat-Flash-Prover

025.2 万星的 AI 视频生成工具

MoneyPrinterTurbo 这个项目在 GitHub 上已经有 5.2 万 Star 了。

是 MoneyPrinter 系列里最火的一个。

它的功能很简单粗暴。

你给它一个主题或者关键词，它就能一键生成一条完整的短视频，包括文案、配音、字幕和背景音乐，全部自动搞定。

支持 12 种以上的大模型接入，OpenAI、DeepSeek、通义千问、Google Gemini 都能用。

语音合成支持 Edge TTS 和 Whisper，字幕的字体、位置、颜色都可以自定义配置。

还支持批量生成，一次出多个版本。

技术架构上是完整的 MVC 模式，同时提供了 Web 界面和 REST API。Docker 一键部署，Windows 也有启动包，上手门槛很低。

如果你在做自媒体，这个工具确实能省不少事。

  * 

    
    
    开源地址：https://github.com/harry0703/MoneyPrinterTurbo

Supermemory 是一个为 AI 提供持久记忆的引擎，目前在 GitHub 上有 1.8 万 Star。

在 LongMemEval、LoCoMo、ConvoMem 三大 AI 记忆基准测试中。

它全部排名第一。

它的核心能力是自动从对话中提取事实，处理时间变更和矛盾信息，过期的内容还会自动遗忘。

它维护的用户画像分为静态档案和动态上下文两部分，单次调用只要 50ms 左右。

搜索方面做了 RAG + Memory 的混合检索，知识库文档和个性化上下文可以同时返回。

PDF、图片、视频、代码都能处理，Google Drive、Gmail、Notion、GitHub 这些平台也支持实时同步。

集成方面支持小龙虾 等 AI Agent 产品，都能直接用。

如果你在开发 AI 应用，需要给 AI 加上长期记忆，这个项目值得一看。

  * 

    
    
    开源地址：https://github.com/supermemoryai/supermemory

04浏览器里的 3D 建筑设计工具

Pascal Editor 是一个完全跑在浏览器里的免费开源 3D 建筑设计编辑器。

目前在 GitHub 上有 4200 多 Star。

你打开网页就能开始设计房子，墙体、楼板、屋顶、门窗这些都能直接拖拽操作。

多楼层管理也做了，堆叠、展开、单独查看都支持。家具门窗放上去之后还有碰撞检测，不会穿模。

技术实现上用了 React Three Fiber + WebGPU 来做 GPU 加速渲染。

布尔运算用 three-bvh-csg 处理门窗开洞。

状态管理用 Zustand，撤销重做支持 50 步历史。

项目保存到云端，还能分享到社区。

最近刚上过 GitHub Trending，单日涨了 2300 多 Star，今天还在更新代码。

对于需要快速做个建筑原型或者室内布局规划的人来说，这个工具挺实用的。

  * 

    
    
    开源地址：https://github.com/pascalorg/editor

05自动化在线赚钱的全能工具集

MoneyPrinterV2 在 GitHub 上拿到了 2.4 万 Star。

定位也是一个自动化在线 make “马内”的全能工具。

它集成了几个核心模块：

  * Twitter 机器人可以自动发推并定时调度

  * YouTube Shorts 自动化用 AI 生成图像当视频素材配上 TTS 语音

  * 联盟营销功能能自动生成 Amazon 推广链接

  * 本地商家挖掘和冷拓展功能

  * 自动搜索商家信息发营销邮件。

V2 相比 V1 做了全面重写，最大的变化是用 AI 生成图像替代了库存视频素材，内容更独特。

脚本生成接入了 gpt4free，Docker 部署也支持了。

所有任务都可以通过 CRON 定时自动执行，基本实现了无人值守的内容生产和推广。

  * 

    
    
    开源地址：https://github.com/FujiwaraChoki/MoneyPrinterV2

06完全离线的知识教育服务器

Project N.O.M.A.D. 最近在 GitHub Trending 上表现很猛。

目前已经有 1.4 万 Star，今天还在更新。

它的概念很有意思：一个离线的知识服务器，打包了 AI 能力、知识库和教育平台，不需要网络就能用。

核心功能包括基于 Ollama + Qdrant 的本地 AI 聊天和文档检索。

通过 Kiwix 提供离线维基百科和医学参考资料，通过 Kolibri 提供 Khan Academy 的课程，还有可下载的离线地图。

安装很简单，一键脚本拉起 Docker Compose 就行。

完全离线运行，零遥测，只有安装的时候需要联网。Web 管理界面基于 AdonisJS 开发，硬件评分和社区排行榜也做了。

3月16日刚上过 Trending，单日涨了 1000 多 Star。

之后连续发了三个版本，迭代速度很快。

适合做家庭自己部署的知识服务器、离网环境下的教育支持，或者注重隐私的本地 AI 环境。

  * 

    
    
    开源地址：https://github.com/Crosstalk-Solutions/project-nomad

07AI 驱动的社区热点研究 Skill

last30days-skill 是一个 Claude Code、Codex CLI 的 SKill，目前在 GitHub 上有将近 5000 Star。

它能自动扫描 Reddit、X、YouTube、Hacker News、Instagram Reels、Polymarket 等平台近 30 天的讨论。

帮你提炼出最新的 prompt 和研究摘要。

工作流程分两步：先发现关键的账号和社区，再深度挖掘具体内容。

去重、评分排序和引用溯源都内置了。

而 YouTube 视频通过 yt-dlp 抓字幕，从 20 分钟的视频里提取高信号信息。

它还有一个 Watchlist 功能，添加关注的主题之后可以定时跟踪动态。

3月密集发了好几个版本，Hacker News、Polymarket、Instagram Reels 这些数据源都是最近加的。

Dave Morin 等人也公开推荐过这个项目。

如果你需要快速了解某个领域最近在讨论什么，这个 Skill 能帮你省大量刷信息源的时间。

  * 

    
    
    开源地址：https://github.com/mvanhorn/last30days-skill
